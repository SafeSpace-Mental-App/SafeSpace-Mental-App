SafeSpace is a mental health support application designed to provide users with accessible tools for mindfulness, mood tracking, and community resources. It helps individuals prioritize their mental well-being through an intuitive interface.

## Features

- **Mood Tracker**: Log daily moods and view trends over time.
- **Mindfulness Exercises**: Guided meditations and breathing techniques.
- **Resource Library**: Curated articles, hotlines, and professional referrals.
- **Community Forum**: Safe space for sharing experiences (coming soon).
- **Personalized Insights**: AI-driven recommendations based on user data.
